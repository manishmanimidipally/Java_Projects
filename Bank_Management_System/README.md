# 🏦 Bank Management System

A **console-based Bank Management System** developed in **Java** using Object-Oriented Programming (OOP) principles. This application allows users to create bank accounts, deposit and withdraw money, check balances, search accounts, display all account details, and delete accounts through a simple menu-driven interface.

---

## 📌 Features

* ✅ Create a new bank account
* 💰 Deposit money
* 💸 Withdraw money
* 📊 Check account balance
* 🔍 Search account by account number
* 📋 Display all accounts
* ❌ Delete an account
* 🚪 Exit the application

---

## 🚀 How to Run

1. Clone this repository.

```bash
git clone https://github.com/your-username/Java_Projects.git
```

2. Open the project in Eclipse or VS Code.

3. Compile the Java files.

```bash
javac *.java
```

4. Run the application.

```bash
java Main
```
