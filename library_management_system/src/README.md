# 📚 Library Management System

A **console-based Library Management System** developed using **Core Java** and **Object-Oriented Programming (OOP)** principles. This application allows users to manage library books through a simple menu-driven interface.

---

## 🚀 Features

* ➕ Add a new book
* 📖 View all books
* 🔍 Search a book by ID
* 📕 Issue a book
* 📗 Return a book
* 🗑️ Delete a book
* 🖥️ User-friendly menu-driven console interface

---
