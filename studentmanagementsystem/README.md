# 🎓 Student Management System

A **console-based Student Management System** developed in **Java** using **Object-Oriented Programming (OOP)** concepts. This project allows users to manage student records through a simple menu-driven interface.

## 🚀 Features

* ➕ Add Student
* 📋 View All Students
* 🔍 Search Student by ID
* ✏️ Update Student Details
* ❌ Delete Student
* 🏆 Display Topper
* 📊 Calculate Average Marks
* 🔤 Sort Students by Name
* 📈 Sort Students by Marks
